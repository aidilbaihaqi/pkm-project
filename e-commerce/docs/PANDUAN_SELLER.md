### 7.2 Cara Membaca Performa

**Skenario 1: Views Tinggi, WA Clicks Rendah**
- **Analisis:** Konten menarik untuk dilihat, tapi tidak memotivasi untuk order
- **Solusi:**
  - Cantumkan harga yang jelas di deskripsi
  - Tambahkan call-to-action: "Order via WA!"
  - Jelaskan keunggulan produk lebih detail

**Skenario 2: Likes Tinggi, Views Rendah**
- **Analisis:** Orang yang tonton suka, tapi jangkauan terbatas
- **Solusi:**
  - Upload lebih sering (2-3x per minggu)
  - Gunakan thumbnail yang eye-catching
  - Post di jam prime time (12:00-13:00, 18:00-21:00)

**Skenario 3: WA Clicks Tinggi**
- **Analisis:** Konten Anda berhasil! ğŸ‰
- **Solusi:**
  - Analisis apa yang berbeda dari konten ini
  - Buat lebih banyak konten serupa
  - Pertahankan gaya dan format yang sama

**Skenario 4: Semua Metrik Rendah**
- **Analisis:** Konten perlu improvement
- **Solusi:**
  - Perbaiki kualitas video/foto (pencahayaan, fokus)
  - Review deskripsi - apakah informatif?
  - Coba kategori atau jenis konten berbeda

### 9.1 Pertanyaan Umum

**Q: Apakah ada biaya untuk menjadi seller?**
A: Tidak ada! Platform ini 100% GRATIS untuk semua seller. Tidak ada biaya pendaftaran, biaya bulanan, atau potongan penjualan.

**Q: Konten saya tidak muncul di feed, kenapa?**
A: Cek hal berikut:
1. Pastikan status konten **"Tayang"** bukan Draft
2. Pastikan profil UMKM sudah lengkap
3. Coba refresh halaman
4. Jika masih bermasalah, konten mungkin sedang direview admin

**Q: Bagaimana cara mengganti video yang sudah diupload?**
A: Video/foto tidak bisa diganti setelah diupload. Anda perlu:
1. Hapus konten yang ada
2. Upload ulang dengan video/foto baru

**Q: Berapa durasi maksimal video?**
A: Tidak ada batasan durasi, tapi kami sangat merekomendasikan **15-60 detik** untuk engagement optimal. Video terlalu panjang cenderung di-skip.

**Q: Bisa upload dari HP?**
A: Ya! Website kami responsive dan bisa diakses dari browser HP (Chrome, Safari, dll). Anda bisa langsung upload video dari galeri HP.

**Q: Pelanggan tidak bisa menghubungi saya via WhatsApp?**
A: Cek hal berikut:
1. Pastikan format nomor benar (62xxx tanpa 0 atau +)
2. Pastikan nomor WhatsApp aktif
3. Coba update nomor di Edit Profil

**Q: Kenapa harus pakai akun Google?**
A: Platform menggunakan Google OAuth untuk keamanan. Ini lebih aman karena:
- Tidak perlu ingat password tambahan
- Keamanan akun dikelola Google
- Login lebih cepat

**Q: Konten saya dihapus admin, kenapa?**
A: Kemungkinan konten melanggar kebijakan platform:
- Konten dewasa/vulgar
- Produk ilegal
- Penipuan/scam
- Spam/clickbait berlebihan
- Hubungi support untuk klarifikasi

### 9.2 Troubleshooting

**Masalah: Tidak bisa login**

| Solusi | Langkah |

| 2 | Clear browser cache dan cookies |

**Masalah: Upload video gagal**

| Solusi | Langkah |
|--------|---------|
| 1 | Cek ukuran file (maks 100 MB) |
| 2 | Cek format file (MP4 atau WebM) |
| 3 | Pastikan koneksi internet stabil |
| 4 | Coba kompres video terlebih dahulu |
| 5 | Refresh halaman dan coba ulang |

**Masalah: Statistik tidak update**

| Solusi | Langkah |
|--------|---------|
| 1 | Klik tombol Refresh di dashboard |
| 2 | Tunggu beberapa menit (data tidak real-time instant) |
| 3 | Reload halaman browser |

| Solusi | Langkah |
|--------|---------|
| 1 | Cek format nomor di profil (harus 62xxx) |
| 2 | Pastikan nomor WhatsApp aktif |

| Solusi | Langkah |
|--------|---------|
| 1 | Tunggu beberapa saat (proses upload) |
| 2 | Refresh halaman |
